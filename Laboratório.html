Laboratório DevOps

Esse site é um laboratório de testes

Os arquivos que compoem esse laboratório estão hospedados no GitHub
o projeto é público e se você quiser ver, fique à vontade.

Como IaC nesse laboratório foi usada uma instância EC2 t2 da AWS
Essa instância está dentro de uma VPC personalizada e tem um Security Group configurado
somente as portas 80 e 8080 estão abertas 
caso você não saiba o que é uma VPC clique aqui

O Jenkins é responsável por criar um Container Docker NGINX personalizado com os arquivos do GitHub
e atualizar a imagem toda vez que o GitHub recebe um commit novo
caso você queira acessa-lo clique aqui
usuário: jenkins
senha: jenkins

O pipeline do Jenkins foi escrito em Bash e Groovy, para acessar clique aqui
O site foi construído com a ajuda do Nicepage
